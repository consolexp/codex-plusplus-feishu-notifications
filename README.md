# Feishu Notifications

Send Feishu notifications when Codex turns complete.

This is a Codex++ tweak for [b-nnett/codex-plusplus](https://github.com/b-nnett/codex-plusplus).

## Install

Copy this repository into your Codex++ local tweaks directory, or install it from the Codex++ Tweak Store after review.

## Manifest

- ID: `io.github.consolexp.feishu-notifications`
- Version: `0.1.0`
- Scope: `both`
- Entry: `index.js`
